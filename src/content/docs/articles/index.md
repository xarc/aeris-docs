# Articles

Several academic works were produced during the development of **ÆRIS** and its underlying simulation engine.  
These publications describe the motivation, design decisions, and educational evaluation of the simulator.

The papers cover topics such as:

- RISC-V assembly education
- interactive simulation tools
- architecture of the simulator core
- educational evaluation in undergraduate courses

---

## Analysis of RISC-V Assembly Language Teaching Using an Interactive Web Simulator  
**JICS**

**Authors:**  
Eduardo Michel Deves de Souza, Nathalia Suzin, Cesar Albenes Zeferino, Douglas Rossi de Melo  
University of Vale do Itajaí (UNIVALI)

**Summary**

This work evaluates the use of a web-based RISC-V simulator in undergraduate computer architecture courses.  
The simulator provides an intuitive browser-based environment for editing, assembling, and executing RV32I assembly programs.

The study compares the tool with **RARS**, a widely used RISC-V educational simulator, using Likert-scale questionnaires applied to students.

**Main findings**

- Strong ratings for usability and interface clarity
- Improved visualization of registers and memory during execution
- Effective support for introductory RISC-V teaching

## Interactive Simulator for RISC-V Assembly Programming  
**SEPOC**

**Authors:**  
Eduardo M. D. Souza, Nathalia Suzin, Cesar A. Zeferino, Douglas R. Melo  
Laboratory of Embedded and Distributed Systems (LEDS) – UNIVALI

**Summary**

This paper presents the design and implementation of an interactive web simulator for the **RV32I instruction set**.

The simulator integrates:

- a code editor
- an assembler
- a processor model
- visualization of registers and memory

It supports both **continuous execution** and **step-by-step execution**, allowing users to observe how instructions modify processor state.

The tool is intended as a didactic resource for students and instructors working with RISC-V assembly programming.

## Simulador Interativo para Programação na Arquitetura de Processadores RISC-V  
**COTB 2025**

**Authors:**  
Eduardo Michel Deves de Souza  
Larissa de Sousa Gouvea  
Cesar Albenes Zeferino  
Douglas Rossi de Melo  
Universidade do Vale do Itajaí (UNIVALI)

**Summary**

This work presents an interactive web-based simulator focused on the **RV32I instruction set architecture**.

The simulator integrates:

- a parser
- an assembler
- a processor model
- an interactive user interface

It enables both **continuous execution** and **instruction-by-instruction debugging**, allowing users to observe register updates and memory changes.

The tool is designed to support teaching activities in computer architecture and embedded systems courses.

## Desenvolvimento do Núcleo de um Simulador Web para a Arquitetura RISC-V  
**SEMICRO**

**Authors:**  
Eduardo M. D. Souza, Thiago H. Rausch, Cesar A. Zeferino, Douglas R. Melo  
Laboratory of Embedded and Distributed Systems (LEDS) – UNIVALI

**Summary**

This paper describes the development of the **core architecture of a RISC-V simulator**.

The simulator core was implemented in **TypeScript** and consists of three main modules:

- code analyzer
- assembler
- processor model

The core is designed to be integrated into a web interface, enabling interactive experimentation with RISC-V assembly programs.

## Simulador Interativo para Estudo de Programação em Linguagem de Montagem na Arquitetura RISC-V  
**IBERCHIP**

**Authors:**  
Eduardo M. D. Souza, Larissa S. Gouvea, Thiago H. Rausch, Cesar A. Zeferino, Douglas R. Melo  
Laboratory of Embedded and Distributed Systems (LEDS) – UNIVALI

**Summary**

This work presents an interactive simulator designed to support the study of assembly programming in the **RISC-V architecture**.

The simulator:

- supports RV32I instructions and pseudo-instructions
- runs entirely in the browser
- was implemented using **TypeScript and Angular**

Experimental evaluation shows that the simulator requires **less memory than comparable tools**, making it suitable even for devices with limited resources.
