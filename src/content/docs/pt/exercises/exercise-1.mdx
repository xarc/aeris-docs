Baseado em Patterson & Hennessy, páginas 54–56. Percorra o programa abaixo instrução por instrução e preencha a tabela, registrando o valor do PC, a instrução executada e o valor dos registradores modificados após cada instrução.

<RiscVExercise
  code={`# C expression: f = (g + h) - (i + j)

.text

    addi t3, zero, 2      # i = 2
    addi t4, zero, 1      # j = 1

    add  t5, t1, t2       # t5 = g + h
    add  t6, t3, t4       # t6 = i + j
    sub  t0, t5, t6       # f = t5 - t6 (result in t0)`}
