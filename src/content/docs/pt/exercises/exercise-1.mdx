## Objetivo

Executar o primeiro exemplo de programa em linguagem de montagem RISC-V.

## Instruções

1. Abra o simulador de linguagem RISC-V.
2. No editor de texto do simulador, transcreva o código abaixo:

```asm
# Exercício 01
# C expression: f = (g + h) - (i + j)

    .text

main:
    addi t1, zero, 4      # g = 4
    addi t2, zero, 3      # h = 3
    addi t3, zero, 2      # i = 2
    addi t4, zero, 1      # j = 1

    add  t5, t1, t2       # t5 = g + h
    add  t6, t3, t4       # t6 = i + j
    sub  t0, t5, t6       # f = t5 - t6 (resultado em t0)
```
